* アプリの概要
- ビンゴゲーム
- しかしただ番号を聞いて穴を空けるのではない
- 通常の抽選とマス開放に加えて、共有イベントゲージが存在する
- 誰かが通常マスを開くたびに共有イベントゲージが増加する
- 共有イベントゲージが満タンになると、次ラウンド開始時に共有イベントが発火する
  
* ユーザーの流れ
1.ルーム準備
  - ホストユーザーがルームを作成
    - このときルームにIDが付与される
  - 参加者がルームIDを入力して入室
  - 全員揃ったらホストユーザーがゲームを開始

2.初期配布
  - 各プレイヤーにビンゴカードを配布
    - 5x5のマスのビンゴカード
  - 各プレイヤーはFREEマスを開いて準備完了にする

3.ラウンド進行
  1. 番号が一つ抽選され、全員に公開される
  2. 共有イベントゲージが満タンなら、抽選後に共有イベントが発火する
    - 全プレイヤーにイベントモーダルが表示される
    - 全プレイヤーがイベント確認を完了すると、通常進行へ戻る
    - イベント完了時に共有イベントゲージはリセットされる
  3. 該当番号があればクリック(タップ)でマスを開く
  4. 該当番号がなければスキップする
  5. 全プレイヤーのアクションが完了したら、ホストが次ラウンドを開始する
