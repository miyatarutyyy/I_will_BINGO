* アプリの概要
- ビンゴゲーム
- しかしただ番号を聞いて穴を空けるのではない
- 引いた手札を活用して自分が有利になるように盤面を変えていく
  
* ユーザーの流れ
1.ルーム準備
  - ホストユーザーがルームを作成
    - このときルームにIDが付与される
  - 参加者がルームIDを入力して入室
  - 全員揃ったらホストユーザーがゲームを開始

2.初期配布
  - 各プレイヤーにビンゴカードを配布
    - 5x5のマスのビンゴカード
  - 手札を3枚引く

3.ラウンド進行
  1. 番号が一つ抽選され、全員に公開される
  2. 該当番号があればクリック(タップ)できるようになる
  3. 各プレイヤーは手札を1枚まで使える
